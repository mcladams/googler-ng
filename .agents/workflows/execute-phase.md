# Phase Execution Workflow

When instructed to "run the next phase":
1. **Read State:** Read `tasks.md`.
2. **Identify:** Find the first Phase where the primary checkbox is `[ ]`.
3. **Execute:** Perform ONLY the bulleted tasks under that specific Phase. Do not look ahead to the next phase.
4. **Update State:** Once the files are successfully modified, update the checkboxes for that Phase in `tasks.md` to `[x]`.
5. **Next stage** Proceed to the next stage and continue through all stages up to stage 8 "Verification - user input required"
