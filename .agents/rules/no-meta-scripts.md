# Refactoring Execution Protocol

When asked to extract, move, or refactor code:
1. **Direct Action:** You must use your native file-editing capabilities to modify, create, and delete files directly.
2. **No Tooling:** DO NOT write Python, Bash, or AST-parsing scripts (e.g., `split.py`, `refactor.sh`) to programmatically parse or split files.
3. **Copy/Paste:** Manually extract the requested classes/functions from the source file and place them in the target file.
